1. 반응형 레이아웃 - Responsive Layout
    - 웹의 레이아웃을 만들 때 사용자의 해상도를 고려
    - 스마트폰이나 태블릿등은 화면이 작기 때문에 가독성이 중요
    - 이런 문제를 해결하는 방법 중 하나
    - 해상도에 따라 가로폭이나 배치를 변경

    @media 키워드로 반응형 레이아웃이 가능
    @media (max-width: 768px) {
        body {
            color: red;
        }
    }
